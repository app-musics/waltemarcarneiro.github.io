Google cria leitor de QR Code a longa distância para Android


O Google está trabalhando em um novo scanner que pode facilitar a leitura de QR codes distantes em smartphones Android. A atualização inclui a capacidade de detectar automaticamente um QR code no quadro da câmera, ampliar a imagem e lê-lo. Embora ainda não esteja à disposição dos usuários de dispositivos Android, a atualização já está disponível nas APIs mais recentes do Google, que podem ser utilizadas por desenvolvedores. Segundo o especialista em Android, Mishaal Rahman, essa nova funcionalidade está disponível na biblioteca ML Kit Barcode Scanning API versão 17.2.0, podendo ser implementada em diversos aplicativos. Ao utilizar essa API, os desenvolvedores economizam tempo, já que os usuários não precisam conceder permissões de acesso à câmera.


Google has updated ML Kit's Barcode Scanning API and the Code Scanner API with auto-zoom, enabling the camera to automatically zoom in to scan barcodes that are far away. The scanner will intelligently detect barcodes and zoom in so the user doesn't have to manually adjust zoom. pic.twitter.com/yZGjJsrgI9

O Google atualizou a API de leitura de código de barras do kit de ML e a API do leitor de código com zoom automático, permitindo que a câmera amplie automaticamente para ler códigos de barras distantes. O scanner detectará códigos de forma inteligente e aumentará o zoom para que o usuário não precise ajustar o zoom manualmente.

O Google utiliza sua própria tecnologia de aprendizado de máquina para interpretar os códigos QR, mantendo a privacidade dos usuários, uma vez que apenas um objeto de código de barras é enviado ao aplicativo.

Acredita-se que essa função de zoom automático possa ser incorporada nos dispositivos que executam o Android 13 ou posterior, já que o scanner de código QR desses dispositivos utiliza a mesma biblioteca ML Kit Barcode Scanning onde essa nova funcionalidade está sendo adicionada.

Há indícios de que a atualização já esteja disponível via GMS (Google Mobile Services), que são as partes proprietárias do Android licenciadas pelo Google para fabricantes selecionados.

A eficácia dessa nova funcionalidade em smartphones Android com câmeras de qualidade inferior ainda não está clara, assim como se ela será disponibilizada em dispositivos que não sejam os próprios modelos Pixel do Google.
